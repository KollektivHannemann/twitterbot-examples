
# Twitterbot Examples

A couple of reactive twitterbot examples for the bots n plots course

## create a virtual environment
    workon botsnplots
    
## Install twitterbot + fix
    pip install git+https://github.com/craftoid/twitterbot

## Alice Bot 
    pip install aiml
